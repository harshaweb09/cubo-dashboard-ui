# CUBO Dashboard UI

![Project Preview](./preview.png)

A modern creative dashboard UI built using semantic HTML, CSS Grid, and Flexbox layouts.

This project was created as part of my frontend practice while learning web development from Sheriyans Coding School Cohort 3.0.

---

## Tech Stack

- HTML5
- CSS3
- CSS Grid
- Flexbox

---

## Features

- Modern dashboard UI design
- Advanced CSS Grid layouts
- Reusable card-based components
- Nested layout systems
- Responsive spacing structure
- SVG & icon integration
- Clean visual hierarchy
- SEO-friendly metadata
- Open Graph meta tags

---

## Concepts Practiced

- CSS Grid architecture
- Grid spanning & placement
- Nested flexbox layouts
- Component-based UI thinking
- Layout decomposition
- Card system design
- Visual hierarchy
- Typography & spacing systems
- Semantic HTML
- Modern dashboard composition
